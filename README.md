# Aegis Vanguard / Эгида Авангарда

Статический интерактивный фан-лоре сайт во вселенной `Transformers: Prime` о затерянном автоботском корабле `Aegis Vanguard`.

Текущая версия проекта собрана без озвучки и делает упор на:

- главы и сцены истории;
- карточки офицеров;
- разделы корабля;
- рядовой экипаж и рабочие палубы;
- журнал миссии;
- визуальную атмосферу корабельного интерфейса автоботов.

## Структура

- `index.html` — каркас сайта.
- `styles.css` — визуальный стиль и адаптивность.
- `script.js` — загрузка данных и интерактивная навигация по истории.
- `data/story.json` — главы, сцены, разделы корабля и mission log.
- `data/characters.json` — главные персонажи экипажа.
- `assets/images/` — корабль, интерьеры, офицеры и общий экипаж.

## Как открыть локально

Откройте проект через локальный сервер, иначе браузер может заблокировать загрузку JSON.

### Вариант через Python

```bash
python -m http.server 8000
```

После этого откройте:

`http://localhost:8000`

## Как добавить новую главу

1. Откройте `data/story.json`.
2. Добавьте новый объект в массив `chapters`.
3. Если нужно, добавьте связанный элемент в `missionLog`.

## Как добавить нового офицера

1. Добавьте нового персонажа в `data/characters.json`.
2. Положите его изображение в `assets/images/`.
3. Используйте его `id` в `speakerId` у нужных реплик.

## Как улучшать визуальную часть

Сайту особенно полезны:

- отдельные полноростовые изображения офицеров без текстовых плашек;
- групповые сцены мостика, ангара, реактора и медотсека;
- изображения обычного экипажа на рабочих палубах;
- кадры, где корабль и интерьер показаны с масштабом.

Менее полезны:

- те же самые карточки, но только с другой рамкой;
- изображения, где много текста уже вшито в арт;
- слишком тесные крупные планы, если они не несут эмоцию или сцену.

## Примечание

Проект не пытается сделать `Aegis Vanguard` сильнее всех во вселенной. История строится на долге, изоляции, выживании и позднем возвращении в войну.
